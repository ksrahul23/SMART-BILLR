# SMART BILLR

A modern, customizable invoice generator built with React and Vite. Create professional invoices with live preview and export to PDF.

## Features

- Dual Theme Support (Light/Dark/System)
- Live Preview
- Auto-Save (LocalStorage)
- Customizable Fields
- Dynamic Calculations
- Multi-Currency Support
- Logo Upload
- Responsive Design
- Keyboard Shortcuts
- Drag & Drop Sections
- Bill History (Supabase) - Logged-in users can automatically track and manage all previously created bills from their dashboard.

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/ksrahul23/SMART-BILLR.git
cd SMART-BILLR
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

## Tech Stack

- React
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase (Auth & Database)
- jsPDF + html2canvas
